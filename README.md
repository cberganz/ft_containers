My implementation of the following containers of the C++ standard template library :
* Vector
* Stack
* Map
* Set

This project includes re-implementation of the following tools :
* RBTree
* iterator_traits
* reverse_iterator
* enable_if
* is_integral
* equal
* lexicographical_compare
* pair
* make_pair
